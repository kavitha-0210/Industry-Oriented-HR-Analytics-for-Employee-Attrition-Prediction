# Industry-Oriented-HR-Analytics-for-Employee-Attrition-Prediction
The HR Analytics project was developed to analyze employee data and predict attrition risk using data science techniques. Exploratory analysis revealed that employees with lower job satisfaction, lower salaries, and higher overtime participation were more likely to leave the organization.

A Logistic Regression model was used to predict employee attrition. The model successfully identified patterns associated with employee turnover and provided useful insights for workforce planning.

The analysis demonstrates that HR analytics can support data-driven decision-making, improve employee retention strategies, and reduce organizational costs associated with employee turnover.
